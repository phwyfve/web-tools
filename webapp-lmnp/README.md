# LMNP Manager

Application React pour la gestion de Location Meublée Non Professionnelle (LMNP).

## Fonctionnalités

- 🔐 Authentification sécurisée
- 🌙 Mode sombre / Mode clair
- 📊 Tableau de bord avec statistiques
- 🏢 Gestion des informations SIREN
- 🏠 Gestion des logements
- ⚡ Suivi de l'usage des logements
- 💰 Gestion des recettes
- 📉 Gestion des dépenses
- 💳 Suivi des emprunts
- 📋 Gestion OGA
- 🧾 Statut fiscal et social

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

L'application sera accessible sur `http://localhost:5173`

## Build

```bash
npm run build
```

## Technologies

- React 19
- TypeScript
- Vite
- React Router
- Tailwind CSS
- Lucide React (icônes)

## Structure du projet

```
src/
├── components/       # Composants réutilisables
├── contexts/         # Contextes React (Auth, Theme)
├── pages/            # Pages de l'application
└── main.tsx          # Point d'entrée
```
